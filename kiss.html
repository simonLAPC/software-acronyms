KISS

Keep It Simple, Stupid

Systems work best if they are kept simple rather than made complicated; therefore, simplicity should be a key goal in design.

  • Easier to maintain and debug.

  • Faster for new developers to understand.

  • Reduces the chance of bugs.

Complex Approach

# Complex: Over-engineered class hierarchy
class AbstractWriter:
    def write(self, data):
        raise NotImplementedError

class FileWriter(AbstractWriter):
    def __init__(self, filename):
        self.filename = filename
    def write(self, data):
        with open(self.filename, 'w') as f:
            f.write(self._format(data))
    def _format(self, data):
        return str(data)

class JsonFileWriter(FileWriter):
    def _format(self, data):
        import json
        return json.dumps(data, indent=2)

# Usage is unnecessarily complex
writer = JsonFileWriter('user.json')
writer.write({'name': 'Alice'})

KISS Approach (Simplified)

# Simple: A straightforward function
import json

def write_json_file(filename, data):
    """Writes a dictionary to a JSON file."""
    with open(filename, 'w') as f:
        json.dump(data, f, indent=2)

# Simple, direct usage
write_json_file('user.json', {'name': 'Alice'})
