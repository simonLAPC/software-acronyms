YAGNI

You Ain't Gonna Need It

A programmer should not add functionality until deemed necessary. Avoid building features on speculation.

  • Prevents wasted development time.

  • Keeps the codebase smaller and cleaner.

  • Focuses on delivering current requirements.

# YAGNI: Implement only what's necessary now
import json

def write_json_file(filename, data):
    """Writes data to a JSON file."""
    with open(filename, 'w') as f:
        json.dump(data, f, indent=2)

def write_text_file(filename, text_data):
    """Writes text to a file."""
    with open(filename, 'w') as f:
        f.write(text_data)
